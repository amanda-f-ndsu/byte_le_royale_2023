==========
Storage
==========

Important information
------------------------
The storage allow you place a ingredient/topping inside to be use later on.
The storage container are very outdated within the Pizzaa parlour as it can only hold one topping at a time.

Instance Variables
------------------

================  =========================== ===================
 Name              Type                        Description
================  =========================== ===================
Item               PizzaState enums            A topping that are store in the storage
================  =========================== ===================
